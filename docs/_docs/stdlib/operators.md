# Operators

- `(+ a b)` - sum
- `(- a b)` - subtract
- `(* a b)` - multiply
- `(/ a b)` - dividy
- `(^ a b)` - raise to the power of
- `(== a b)` - equal
- `(!= a b)` - not equal
- `(> a b)` - greater
- `(< a b)` - less
- `(>= a b)` - greater or equal
- `(<= a b)` - less or equal
- `(not args...)` - returns true if all statements are false
- `(and args...)` - returns true if all statements are true
- `(or args..)` - returns true if any statement is true
